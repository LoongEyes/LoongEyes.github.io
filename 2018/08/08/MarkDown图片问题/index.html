MarkDown图片问题

发表于 2018-08-08 | 更新于 2018-08-10 | 分类于 搭建博客

使用Github+hexo搭建博客之后，markdown写作手法确实会节省不少时间与工作量。但是，markdown的图片功能确实不尽如人意，本地文档上传至Github后，无法连接本地路径图片。

构建Github图片仓库

新建仓库

登陆Github

记录一下新建仓库的clone地址

本地clone仓库

本地使用git bash将仓库clone到本地

本地相应目录中会生成仓库名称的文件夹（此处为Test）

添加图片文件

进入Text目录，手动添加图片文件

将本地内容上传至Github仓库

MarkDown使用仓库图片

访问Github仓库，进入图片页面，右击复制图片，将图片链接加入MarkDown文档中，则文档会包含仓库中的图片，而不会依赖于本地图片资源。
