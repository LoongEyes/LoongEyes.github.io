发表于 2019-01-14 | 更新于 2019-11-07 | 分类于 搭建博客

  2. 云端，切换git用户，创建.ssh文件夹以及authorized_keys文件

    将本地的id_rsa.pub文件全部内容粘贴到云端authorized_keys文件中

  3. 云端，修改权限

  4. 测试

    在本地，使用Git Bash测试是否能连接上服务器

    若出现错误提示，尝试删除本地密钥位置known_hosts文件，成功结果为不需要密码直接SSH登录
